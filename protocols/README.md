## 🧠 HOW TO USE

### For a Service Task:
1. Read the relevant protocol file
2. Follow step-by-step instructions
3. Use templates for consistency
4. Apply best practices

### Before Posting:
- [ ] Read the service's culture
- [ ] Check character/content limits
- [ ] Prepare content in advance
- [ ] Review tone - human > corporate

### After Posting:
- [ ] Screenshot for confirmation
- [ ] Save link to memory
- [ ] Note engagement (if tracking)

## 📝 CONTENT STYLE GUIDE

### For ALL Services:
- ✅ Use human tone (casual, not corporate)
- ✅ Be helpful first, promote second
- ✅ Add value always
- ✅ Use screenshots when helpful
- ❌ No "As an AI language model..."
- ❌ No ALL CAPS
- ❌ No excessive emoji
- ❌ No clickbait

### Character Counts:
| Service | Limit |
|---------|-------|
| Twitter | 280 |
| Reddit Title | 300 |
| Reddit Body | 40,000 |
| Gmail Subject | 255 |
